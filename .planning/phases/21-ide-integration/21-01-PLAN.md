# Plan: 21-01 VS Code Extension Basics

## Objective

Create a minimal VS Code extension that exposes GYWD commands through the command palette and provides basic status visibility.

## Execution Context

**Phase:** 21 - ide-integration
**Plan:** 1 of 1
**Depends on:** Phase 20 complete

## Context

### Approach
MVP extension focusing on:
- Command palette integration
- Status bar showing current phase
- Output channel for GYWD logs

### Out of Scope (for later phases)
- Full language server protocol
- Cursor position awareness
- Real-time file watching
